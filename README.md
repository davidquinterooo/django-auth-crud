# Django Task - CRUD y Auth

Una aplicación web robusta desarrollada con **Django** y **Bootstrap**, diseñada para gestionar tareas personales de manera eficiente. El sistema permite a los usuarios autenticados tener un control total sobre su lista de tareas en un entorno seguro.


## Características Principaless

- **Autenticación Completa:** Registro de usuarios, inicio de sesión y cierre de sesión.
- **Operaciones CRUD:** - **Guargar:** Agregar nuevas tareas con descripción.
  - **Tareas pendientes y completadas:** Visualizar la lista de tareas pendientes y completadas.
  - **Actualizar:** Editar detalles de las tareas o marcarlas como completadas.
  - **Eliminar:** Eliminar tareas de forma permanente.
- **Seguridad:** Protección de rutas (solo usuarios logueados pueden gestionar sus propias tareas).
- **Interfaz Responsiva:** Diseño adaptativo gracias a Bootstrap, optimizado para móviles y escritorio.
